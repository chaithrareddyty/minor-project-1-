# 🧮 Smart Calculator

A Smart Calculator built using Python that performs basic mathematical operations with a simple and user-friendly interface. This project is designed to strengthen Python programming fundamentals, including functions, conditional statements, loops, and exception handling.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧠 Smart input validation
- ⚠️ Error handling (e.g., division by zero, invalid input)
- 🔄 Menu-driven interface
- 💻 Easy to use and beginner-friendly

---

## 🛠️ Technologies Used

- Python 3
- Google Colab / Jupyter Notebook
- Standard Python Libraries

## ▶️ How to Run

### Option 1: Google Colab

1. Open the notebook in Google Colab.
2. Click **Runtime → Run All**.
3. Follow the on-screen instructions.

### Option 2: Jupyter Notebook

1. Install Jupyter Notebook.
2. Open the project folder.
3. Run all cells.
